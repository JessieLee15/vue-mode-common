# vue-mode-common

一个vue项目架子demo
* 管理工具：vue-cli 4.0
* 技术栈：
  * vue全家桶（vue vue-router ）
  * axios
  * SPA
  * element-ui（按需引入）

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
